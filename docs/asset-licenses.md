# Commercial Visual Asset Notes

This product UI should only use assets that are safe for commercial product work.

## Active in the app

- Lucide: base UI icon system. License: ISC. Source: https://lucide.dev/license
- Reicon React: Agent, routing, analytics, export, and review icons. License: MIT. Source: https://reicon.dev
- Motion for React: page, card, and state transitions. License: MIT. Source: https://motion.dev
- unDraw SVG assets downloaded to `public/illustrations/`: `undraw-content-team.svg`, `undraw-charts.svg`, `undraw-ai-data-extraction.svg`, `undraw-setup-wizard.svg`, `undraw-team-assignment.svg`. License: commercial use allowed without attribution. Source: https://undraw.co/license

## Approved expansion sources

- Iconoir: open-source icon library for product, content, and marketing metaphors. Source: https://iconoir.com
- Reicon: MIT icon library with React and Figma support. Source: https://reicon.dev
- unDraw: illustration source for empty states and onboarding-style product visuals. Commercial use allowed without attribution under its own license. Source: https://undraw.co/license
- SketchValley: illustration source for dashboard and operating-state visuals. Commercial use allowed without attribution per its license page. Source: https://sketchvalley.com/about/

## Product rule

Do not add stock illustrations, icon packs, Lottie files, or generated assets unless their license is recorded here first.
